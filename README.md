
## 历史数据
Akshare抓取数据（日线），使用splite3进行存储在本地，避免重复抓取导致数据获取失败。
时线运行时获取当天，判断交易点。

## CNN卷积判断趋势
CNN适用于图像类数据，能很好的进行局部特征提取；并且相对RNN神经网络来说其运行速度更快。预计使用该网络阶段性预测数据趋势，进行初步目标筛选，避免后续agent运行时间过长，消耗过多资源。

- 输入：时间窗口长度内的多个数据指标：基础指标`open`, `close`, `low`, `high`, `amount`，技术指标暂时添加额外的17种
- 标签：预测标签为选取时间窗口外紧接着的第一天log收益值。

### CNN数据抓取
`CNNDataset`抓取数据，抓取数据按照`code`分类存放。
- 函数输入：起始日期，终止日期，允许延迟，原始数据(前复权)
- 函数使用：根据起始日期至终止日期的天数获取等长的数据，自动跳过法定休息日与节假日。也就是当指定日期范围内天数不足时，会自动向前寻找合适的起始日期。由于该函数内需要计算30天移动均线，所以额外向前寻找30天。函数实例化后，执行initial开始抓取数据
- 函数返回：完成抓取后按索引取值返回指定日期范围天数的所有`code`原始数据，以及标签值每天相对与上前一天的log收益数据。

### 训练，验证及测试数据生成
根据`CNNDataset`获得的数据，还需要进一步进行时间切片，依据限定日期范围获取所有数据，而后将数据随机按某个固定时间窗口采样。采样数据需要按行进行Z-score标准化。
分别生成train Data, valid Data, test Data.
